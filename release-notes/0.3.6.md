## Adaptive Practice 0.3.6

This release focuses on making high-skill practice feel appropriately challenging, especially for Linux command notes.

### Improved

- Tightened the target difficulty mix so high-skill notes stop receiving easy-heavy daily sessions.
- Added stricter Linux/shell calibration that demotes command recall, shallow option comparisons, simple predictions, and fake-hard command-choice prompts.
- Added high-skill Linux retry checks for hard-question depth, subtopic coverage, and shell-mechanic diversity.
- Improved prompt guidance so advanced shell questions require construction, debugging, output/state prediction, redirection/quoting/permission reasoning, or multi-condition tracing.
- Added a provider-agnostic generation loop that retries under-challenging or underfilled batches before accepting them.

### Fixed

- Repaired non-JSON fenced provider output before parsing, including language-tag mistakes that previously broke question generation.
- Preserved harder surplus questions during flow balancing so stronger replacements are not discarded too early.
- Expanded source-topic reconciliation for Linux/shell aliases so generated questions still attach to the right note.

### Tests

- Added regression coverage for high-skill Linux sessions, fake-medium questions, weak hard labels, repetitive subtopics, mixed-topic coverage, and the current Linux Commands note.
