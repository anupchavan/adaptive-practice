## Fixed

- **"true" section headings in settings on Obsidian 1.13.** The declarative group schema takes the heading *text* in the `heading` field; 0.6.0 passed a boolean, so every group rendered a literal "true" heading and logged `duplicate setting key "group:true"` on every open. Groups now carry their real names and the console stays quiet.

## Changed

- **The topic picker now looks and behaves like a native settings list** (the hotkeys-panel treatment): a proper search field with built-in clear button, filter chips (All / Due / New / Low skill / PDFs) in the native chip style, the course/folder dropdown beside them, and each note as a real settings row — checkbox on the left, title with PDF/folder badges, path and aliases as the description, skill and due status on the right. Rows and inputs are built from Obsidian's own components (Setting, SearchComponent, DropdownComponent), so themes style them for free.
- **License changed to MIT.** The plugin is fully open source, effective now.
