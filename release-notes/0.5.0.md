## Added

- **Analytical moves engine.** The reasoning patterns behind great problem-book questions aren't magic, they're *moves* — reusable maneuvers like approximation breakdown (put the learner where a reflexive idealization stops holding), false symmetry (two views that look interchangeable until a deeper level splits them), necessary-vs-possible (what can you *certainly* conclude?), minimal pair (two setups differing in one feature with different outcomes), limiting case, counterexample hunt, flawed argument, hidden assumption, and ten more. Generation now carries this 18-move catalog and instructs the model to pick the move the material genuinely supports and instantiate it on your note — instead of "write a hard question", which regresses to competent-but-flat. The moves key off *structural* features of the material (a rule with assumptions, a claimed universal, steps that might not commute), never a subject, so the same catalog drives code, math, medicine, law, history, or physics notes. It lives in the cached system prompt, so it costs almost nothing per session.
- **Deep authoring (optional).** A second, adversarial pass over each batch's medium and hard questions: the model attacks its own drafts — can a test-wise learner shortcut the reasoning? is the trap actually tempting and actually wrong? does "certainly" truly hold? do length or wording give the answer away? — and re-emits sharpened rewrites. This roughly doubles tokens per session, so it is off by default. The pass is fail-safe: rewrites the model drops or mangles keep their originals, a failed request keeps the whole batch, and the sharpened set still goes through blind answer verification before it reaches you.
